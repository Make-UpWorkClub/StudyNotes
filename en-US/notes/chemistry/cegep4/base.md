# base

Arrhenius: hydroxide producer
Bronsted-Lowry: proton *acceptor*
Lewis: electron-pair *donor*
`Ant.` [[acid]]

> [!abstract] conjugate base
> Acid that gives proton to become base

> [!important] Stronger base = weaker conjugate acid

## Strength

**Strong** base: base that *fully* dissociates
**Weak** base: base that *partially* dissociates

Strong bases:

-  $\ce{OH^-}$ bases
	- 1A group: $\ce{LiOH, NaOH, KOH}$
	- 2A group: $\ce{Ca(OH)2, Ba(OH)2, Sr(OH)2}$ (low solubility)
