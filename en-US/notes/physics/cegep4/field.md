# field

Function that maps every point in space to a value

Types:

- Scalar: scalar value
- Vector: vector value

Representations:

- Heatmap: grid of colors or symbols varying in size
- Vector map: grid of vectors
- Field line: continuous vector lines tangent to field vectors
	- Lines never cross
	- Density represents magnitude
