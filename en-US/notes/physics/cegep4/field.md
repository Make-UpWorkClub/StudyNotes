# field

Function that maps every point in space to a value

Types:

- Scalar: scalar value
- Vector: vector value

Representations:
	
- Vector map: grid of vectors
- Field line: continuous vector lines tangent to field vectors
	- Lines never cross
	- Density represents magnitude
- Heatmap: grid of colors or symbols varying in size
- Contour map: lines on which the field is constant and between which the field difference is constant
- Elevation graph: 3D visualisation of scalar field

> [!abstract] uniform field
> Field with same magnitude and direction
