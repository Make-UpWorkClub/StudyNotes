# Lois de Kirchhoff

[[Électricité]]

|     |           Série            |                      Parallèle                      |
|:---:|:--------------------------:|:---------------------------------------------------:|
|  R  | $R_{eq} = R_1 + R_2 + ...$ | $R_{eq} = \cfrac1{\frac1{R_1} + \frac1{R_2} + ...}$ |
|  U  |  $U_T = U_1 + U_2 + ...$   |               $U_T = U_1 = U_2 = ...$               |
|  I  |  $I_T = I_1 = I_2 = ...$   |               $I_T = I_1 + I_2 + ...$               |
