# Quantitative Data

[[Data]] recorded on a numerical scale
Opposed to [[qualitative data]]

## Types

### Discrete

Possible values are finitely or infinitely *countable*

### Continuous

Possible values are *uncountable*, i.e. can take any value in an interval
