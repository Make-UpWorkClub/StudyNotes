# Frequency

## Types

### Class frequency

Number of individuals in a particular class
`Sym.` $n_i$

### Class relative frequency

Class frequency divided by sample size
`Sym.` $f_i$

### Cumulative relative frequency

Proportion of **ordered** data until a specific value

$$
F_i = \sum_{ k = 1 }^{ i } f_k
$$
