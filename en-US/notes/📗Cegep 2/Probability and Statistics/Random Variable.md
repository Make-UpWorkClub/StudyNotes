# Random Variable

Value assigned to each [[Simple Event|sample point]]

## Types

### Discrete

Possible values are finite

`Sym.` $X$
Possible value denoted as $x$

> [!abstract] Support
> Set of all possible values of $X$
> `Sym.` $S(X)$

> [!example] Sum of two dice rolls

### Continuous

Can take any value in an interval
