# base

$B = \{ \vec{v}_{1}, \vec{v}_{2}, \dots, \vec{v}_n \}$ est une base de $\mathbb{R}^n$ si

1. $B$ est [[dépendance linéaire|linéairement indépendant]].
2. Tout vecteur $\vec{w} \in \mathbb{R}^n$ peut s'écrire comme une [[combinaison linéaire]] de vecteurs dans $B$.
   $$
  \vec{w} \in \mathrm{Vect}(B) = \mathbb{R}^n
  $$

Dans $\mathbb{R}^n$, $B$ est une base si et seulement si $B$ est linéairement indépendant et $|B| = n$.
