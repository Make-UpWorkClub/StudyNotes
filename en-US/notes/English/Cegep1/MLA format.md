# MLA Format

- Double spaced
	- Use 2x spacing instead of actually inserting newlines
- 12 pt standard font
- Content:
	- Indented
- Bibliography:
	- Reverse indented

```html
											<last-name> <page>
<author>
<instructor>
<course>
<date>

							<title>

<content>

						  Works Cited

<bibliography>
```

## Short citation

End of sentence:  `(<author> <page-or-line>)`

> [!example]
> (Melville 70)

## Bibliography

```html
<authors>. <title-of-source>. <title-of-container>, <other-contributors>, <version>, <number>, <publisher>, <publication-date>, <location>.
```
